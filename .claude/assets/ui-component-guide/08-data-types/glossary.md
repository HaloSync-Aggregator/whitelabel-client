# Glossary

## Fare Related

| Term | Description |
|------|-------------|
| Base Fare | Pure airfare excluding taxes |
| Tax | Various taxes including fuel surcharge, airport tax, etc. |
| Total | Base fare + taxes |
| Total Payment Amount | Airfare + ancillary services total |

---

## Time Related

| Term | Description |
|------|-------------|
| Payment Deadline | Booking will be cancelled if not paid by this time |
| Ticketing Deadline | Ticket must be issued by this time |
| Fare Guarantee Deadline | Current fare is guaranteed until this time |

---

## Booking Related

| Term | Description |
|------|-------------|
| PNR | Airline booking reference number |
| Ticketing | Issuing the actual airline ticket after booking |
| VOID | Cancelling a ticketed ticket on the same day |
| REFUND | Refunding a ticketed ticket |

---

## Journey Related

| Term | Description |
|------|-------------|
| Direct | Non-stop flight without layovers |
| Connecting | Flight with intermediate airport stops |
| Transfer | Changing to a different aircraft at a layover point |
| Codeshare | When the marketing carrier and operating carrier are different |

---

## Seat Related

| Term | Description |
|------|-------------|
| Economy | Most basic cabin class |
| Premium Economy | Wider seats and enhanced service compared to economy |
| Business | Upper class for business travelers |
| First | Highest service class |
| Fare Type | Service variations within the same cabin class (Light, Standard, Flex, etc.) |

---

## Passenger Related

| Term | Description |
|------|-------------|
| Adult | Passenger aged 12 years and older |
| Child | Passenger aged 2 to under 12 years |
| Infant | Passenger under 2 years |
| Primary Contact | Contact information applied to all passengers |

---

## Ancillary Service Related

| Term | Description |
|------|-------------|
| Seat Selection | Pre-assigning a preferred seat |
| Checked Baggage | Luggage stored in the cargo hold |
| Carry-on Baggage | Luggage carried directly into the cabin |

---

## Loyalty Program Related

| Term | Description |
|------|-------------|
| Mileage | Airline loyalty points |
| Mileage Number | Loyalty program membership number |
| Promo Code | Airline discount code |
