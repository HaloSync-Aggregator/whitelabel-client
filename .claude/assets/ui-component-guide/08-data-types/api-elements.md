# Data Display Guide

## Date/Time Display Formats

| Usage | Format | Example |
|-------|--------|---------|
| Date | DD/MMM/YYYY | 15/JAN/2025 |
| Time | HH:mm | 14:30 |
| Date+Time | DD/MMM/YYYY HH:mm | 15/JAN/2025 14:30 |
| Flight duration | HHh MMm | 06h 20m |

---

## Airport/City Display

| Format | Example |
|--------|---------|
| Airport code | ICN, CDG, SIN |
| Terminal | T1, T2 |
| Segment display | ICN → CDG |

---

## Amount Display

| Format | Example |
|--------|---------|
| Default | 1,234,500 KRW |
| Increased amount | +50,000 (red) |
| Discounted amount | -10,000 |

---

## Flight Number Display

| Format | Example |
|--------|---------|
| Carrier + flight number | KE0123 |
| 4-digit display | Pad with leading zeros if flight number is short |

---

## Baggage Display

| Format | Example |
|--------|---------|
| Piece count | 1PC, 2PC |
| Weight | 23KG, 30KG |
| Not included | Not included |

---

## Passenger Name Display

| Format | Example |
|--------|---------|
| English name | HONG/GILDONG |
| English name + type | HONG/GILDONG (ADT) |

---

## Title Display

| Code | Meaning |
|------|---------|
| MR | Adult male |
| MS | Adult female |
| MSTR | Male child |
| MISS | Female child |

---

## Journey Type Display

| Format | Example |
|--------|---------|
| Direct | Direct |
| Connecting | 1 stop, 2 stops |

---

## Codeshare Display

- When the marketing carrier and operating carrier are different
- Displayed as "Codeshare OZ" format
