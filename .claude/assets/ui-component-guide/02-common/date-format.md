# Date/Time Display Formats

## Date Display

| Usage | Format | Example |
|-------|--------|---------|
| Departure/arrival date | DD/MMM/YYYY | 21/FEB/2024 |
| Date+time | DD/MMM/YYYY HH:mm | 21/FEB/2024 16:11 |
| Date of birth | DD/MMM/YYYY | 01/MAR/1990 |
| Passport expiry | DD/MMM/YYYY | 31/DEC/2030 |
| Deadline | YYYY-MM-DD HH:mm:ss | 2024-05-16 23:59:00 |

## Time Display

| Usage | Format | Example |
|-------|--------|---------|
| Departure/arrival time | HH:mm | 16:30 |

|--------|-------|
