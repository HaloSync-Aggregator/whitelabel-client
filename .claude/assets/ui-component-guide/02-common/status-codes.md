# Status Display

## Booking Status

| Status | Meaning |
|--------|---------|
| Booking Complete | Booking has been successfully completed |
| Cancelled | Booking has been cancelled |

---

## Ticket Status

| Status | Meaning |
|--------|---------|
| TICKETED | Airline ticket issued |
| ISSUED | Ancillary service issued |
| VOIDED | Ticket voided |
| REFUNDED | Refund completed |
| EXCHANGED | Reissue completed |

| Status | Meaning |
|--------|---------|

| Confirmation Pending | Awaiting payment |
| Confirmed | Purchase completed |
